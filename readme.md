# Bangalon Web Apps

Ini adalah sebuah aplikasi sistem informasi pengelolaan kios penjualan air galon dan gas elpiji.

## Sistem ini menggunakan

- PHP 7.4
- JS
- MariaDB
- SB-Admin2
